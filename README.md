# MultiWatch
App for watching multiple streams at the same time. 

#### Current features:
- Supported platforms: Twitch
- Video functionality:
   - Move
   - Scale
   - Pin (Maximize)
- Chat functionality:
   - Show / Hide
   - Select chat (when multiple streams opened)
   - Change width (up to 50% of the screen)
- Additional features:
   - Show / Hide top bar
   - Restore previous setup (localStorage)
